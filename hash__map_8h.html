Insert a new key-value pair into the hash map.

Adds a new entry to the hash map. If the key already exists, returns HASH_MAP_DUPLICATE_KEY and does not modify the map. Uses the registered hash function to determine the bucket and adds to the collision chain. Automatically checks load factor and may trigger resizing.

Parameters
mapHash map to insert into
keyKey pointer (ownership transferred to map if successful)
valueValue pointer (ownership transferred to map if successful)
Returns
HASH_MAP_OK on success
HASH_MAP_DUPLICATE_KEY if key already exists (entry not added)
HASH_MAP_ERROR if memory allocation fails
Note
If insertion fails, caller retains ownership of key and value
If successful, map owns key and value (freed with registered functions)

Insert a new key-value pair into the hash map.

Parameters
mapHash map
keyKey to insert
valueValue to associate with key
Returns
HASH_MAP_OK on success, error code on failure
